HTML Notes

Semantic HTML Structure

Posted: June 10, 2025

Proper semantic markup improves accessibility and SEO:

<article>
    <header>
        <h1>Article Title</h1>
    </header>
    <section>
        <p>Main content here...</p>
    </section>
    <footer>
        <p>Article metadata</p>
    </footer>
</article>

Accessible Forms

Posted: June 15, 2025

Always pair labels with inputs:

<label for="username">Username:</label>
<input type="text" id="username" name="username">

Use fieldset for grouped inputs:

<fieldset>
    <legend>Shipping Method</legend>
    <input type="radio" id="standard" name="shipping">
    <label for="standard">Standard</label>
    <input type="radio" id="express" name="shipping">
    <label for="express">Express</label>
</fieldset>

CSS Notes

CSS Grid Layout

Posted: June 5, 2025

Basic grid template columns:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

Grid area placement:

.item {
    grid-column: 2 / 4;
    grid-row: 1;
}

Custom Properties

Posted: June 12, 2025

Define variables in root:

:root {
    --primary-color: #ff6600;
    --spacing-unit: 1rem;
}

Use them anywhere:

.button {
    background: var(--primary-color);
    padding: var(--spacing-unit);
}

DOM Manipulation

Querying elements:

// Single element
const header = document.querySelector('header');

// Multiple elements
const buttons = document.querySelectorAll('.btn');

Event listeners:

button.addEventListener('click', () => {
    console.log('Button clicked!');
});
